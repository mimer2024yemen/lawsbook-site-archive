# lawsbook-site-archive

This repository contains a public mirror/archive extracted from the live site `https://lawsbook.lovable.app`.

## Contents
- Static HTML/CSS/JS assets
- Publicly reachable media and fonts captured during extraction
- `lawsbook_site_public.zip` compressed archive
- `file_list.txt` file inventory
- `manifest_summary.json` extraction summary
- `wget.log` fetch log

## Notes
This is a public-site extraction/mirror of the deployed frontend and linked public assets. It does not include private backend services, secrets, databases, or unpublished source code.
